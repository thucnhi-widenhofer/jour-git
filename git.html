Commande Description
git config Pour configurer les préférences de l’utilisateur.
git init Créer un nouveau dépôt GIT.
git add Ajouter des fichiers à l’index.
git clone Vérification des dépôts, distant ou local.
git commit Valider les modifications apportées au HEAD
git status Affiche la liste des fichiers modifiés ainsi que les fichiers qui doivent encore être ajoutés ou validés.
git push Envoyer les modifications locales apportées à la branche associée
git checkout Créer des branches ou basculer entre elles.
git remote Se connecter à un dépôt distant.
git branch Répertorier, créer ou supprimer des branches.
git pull Fusionner toutes les modifications présentes sur le dépôt distant dans le répertoire de travail local
git merge Fusionner une branche dans la branche active.
git diff Visualiser les conflits d’un fichier
git tag Marquer des commits spécifiques avec des poignées simples.
git log Génère le log d’une branche.
git reset Réinitialiser l’index et le répertoire de travail à l’état du dernier commit
git rm Supprimer des fichiers de l’index et du répertoire de travail.
git stash Enregistre les changements qui ne doivent pas être commit immédiatement.
git show Afficher des informations sur tout fichier git.
git fetch extraire tous les fichiers du dépôt distant qui ne sont pas actuellement dans le répertoire de travail local.
git ls-tree afficher un fichier arborescent avec le nom et le mode de chaque élément.
git cat-file À l’aide de la valeur SHA-1, affichez le type d’un fichier.
git grep Rechercher dans les arbres de contenu des expressions et / ou des mots.
gitk Interface graphique du dépôt local.
gitinstaweb Un serveur Web peut être exécuté par interface avec le dépôt local.
git gc Optimiser le dépôt en supprimant les fichiers inutiles.
git archive Créer un fichier zip ou tar contenant les composants d’un arbre du dépôt.
git prune Les fichiers qui n’ont pas de pointeurs entrants seront supprimés.
git fsck Vérification d’intégrité du système de fichiers git.
git rebase Réapplication des commits sur une autre branche.
